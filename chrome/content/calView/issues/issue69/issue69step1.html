descriptionThe Tabulator project is a generic data browser. Using outline and table modes, it provides a way to browse RDF data. The tabulator is a set of open source Javascript sripts which can run in any web page to add data browsing. It currently runs with Firefox, and requires Firefox preferences to be set -- see the tabulator help page. The Tabulator is open source under the W3C software license.
